figure.py module

figure.plot_validation(x, y, data_range, xlabel, ylabel, title)

Plot the model validation graph and set the format automatically

Parameters:

  • x (list) – data for x-axis, usually it’s experimental result

  • y (list) – data for y-axis, usually it’s simulation result

  • data_range (list) – the range for x and y axis

  • xlabel (string) – x label of the plot

  • ylabel (string) – y label of the plot

  • title (string) – title of the plot. If title=”off”, there’s no title

figure.plot_variable(x, y, xlabel, ylabel, title, xlim, ylim, form)

Plot the x-y figure and set the format automatically

Parameters:

  • x (list) – data for x-axis

  • y (list) – data for y-axis

  • xlabel (string) – x label of the plot

  • ylabel (string) – y label of the plot

  • title (string) – title of the plot. If title=”off”, there’s no title

  • xlim (list) – range for x-axis. If xlim=[0,0], the default range is used

  • ylim (list) – range for y-axis. If ylim=[0,0], the default range is used

  • form (char) – dot form

figure.plot_variable_legend(x, y, legend, xlabel, ylabel, title, xlim, ylim, form)

Plot the x-y plot with multiple legend and set the format automatically

Parameters:

  • x (list) – data for x-axis

  • y (list) – data for y-axis

  • legend (list) – legends for each line

  • xlabel (string) – x label of the plot

  • ylabel (string) – y label of the plot

  • title (string) – title of the plot. If title=”off”, there’s no title

  • xlim (list) – range for x-axis. If xlim=[0,0], the default range is used

  • ylim (list) – range for y-axis. If ylim=[0,0], the default range is used

  • form (char) – dot form

figure.plot_variable_time(variable_time, ylabel, title, ylim, form, interval)

Plot the x-t figure and set the format automatically

Parameters:

  • variable_time (variable_time) – the time-dependent variable to be plotted

  • ylabel (string) – y label of the plot

  • title (string) – title of the plot. If title=”off”, there’s no title

  • ylim (list) – range for y-axis. If ylim=[0,0], the default range is used

  • form (char) – dot form

  • interval (float) – the time step of the variable_time (in unit of hour)

figure.plot_variable_time_legend(variable_time_list, legend, ylabel, title, ylim, form, interval)

Plot the x-t figure with multiple legends and set the format automatically

Parameters:

  • variable_time (list) – the list of the time-dependent variables to be plotted

  • legend (list) – legend for each lines

  • ylabel (string) – y label of the plot

  • title (string) – title of the plot. If title=”off”, there’s no title

  • ylim (list) – range for y-axis. If ylim=[0,0], the default range is used

  • form (char) – dot form

  • interval (float) – the time step of the variable_time (in unit of hour)

figure.twin_plot(variable_time1, variable_time2, ylim1, ylim2)

Plot two time-dependent variables on different y-axis scale and set the format automatically

Parameters:

  • variable_time1 (variable_time) – first time-dependent variable

  • variable_time2 (variable_time) – second time-dependent variable

  • ylim1 (list) – the range of y-axis limit for the first time-dependent variable

  • ylim2 (list) – the range of y-axis limit for the second time-dependent variable
