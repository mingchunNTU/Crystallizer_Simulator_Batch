Welcome to Batch_Crystallizer_Simulator’s documentation!

This program simulates the batch cooling crystallizer with size independent nucleation rate and growth rate kinetic. The crystal size distribution is solved through method of moment and coordinate transformation proposed by Vollmer and Raisch (1). The simulation result including final crystal size distribution and growth rate trajectory can be exported as csv file for further data processing.

Ref:(1) Vollmer, U.; Raisch, J. Control of Batch Cooling Crystallization Processes Based on Orbital Flatness. International Journal of Control 2003, 76 (16), 1635–1643. https://doi.org/10.1080/00207170310001626419.
